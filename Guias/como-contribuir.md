# Como contribuir

Contribuir para o TrilhaDoDev é uma ótima forma de praticar e aprender colaborando com outras pessoas.

## Antes de começar
- Leia o `CONTRIBUTING.md` do projeto.
- Verifique se já existe um conteúdo semelhante.
- Use o `TEMPLATE.md` para manter o padrão.

## Passo a passo
1. Fork o repositório no GitHub.
2. Clone para sua máquina com `git clone`.
3. Crie uma branch nova para a sua contribuição.
4. Faça as mudanças e adicione commits claros.
5. Abra um Pull Request explicando o que mudou.

## Tipos de contribuição
- Adicionar ou melhorar conteúdo.
- Corrigir erros de digitação ou formatação.
- Atualizar exemplos e exercícios.
- Organizar a estrutura de pastas e documentação.

## Boas práticas
- Mantenha o estilo do projeto.
- Não crie arquivos desnecessários.
- Teste o conteúdo antes de enviar.
- Seja gentil e explique suas mudanças no PR.

## Depois da contribuição
- Acompanhe comentários e sugestões.
- Faça ajustes se alguém pedir revisão.
- Celebrate! Sua ajuda melhora o projeto para todo mundo.
