## O que é o TrilhaDoDev?

O TrilhaDoDev é um projeto **open source** de educação tecnológica colaborativa. Aqui você encontra trilhas de aprendizado estruturadas para diversas áreas do desenvolvimento de software — organizadas, progressivas e criadas por pessoas da comunidade.

Mais do que um repositório de materiais, é um lugar onde você pode **aprender e ensinar ao mesmo tempo**, acumulando experiência real de colaboração em projetos open source.

## Como contribuir

O TrilhaDoDev cresce com a comunidade. Você pode contribuir:

- **Melhorando** conteúdo existente (exemplos, exercícios, explicações)
- **Corrigindo** erros ou informações desatualizadas
- **Criando** novos módulos dentro de uma trilha existente
- **Propondo** uma nova trilha via Issue

**Antes de qualquer coisa, leia:**
1. [`CONTRIBUTING.md`](./CONTRIBUTING.md) — Regras e fluxo de trabalho
2. [`TEMPLATE.md`](./TEMPLATE.md) — Como estruturar seu conteúdo

---

## Por que participar?

Contribuir com o TrilhaDoDev é uma experiência real de trabalho em time:

- **Portfólio:** Histórico de commits e Pull Requests visível no seu GitHub
- **LinkedIn:** Experiência em projeto open source de educação tecnológica
- **Aprendizado:** Ensinar é a melhor forma de fixar o conhecimento
- **Comunidade:** Reuniões quinzenais e contato direto com outros devs
- **Flexibilidade:** Contribua no seu ritmo — 20 a 30 minutos já fazem diferença

---

## Contato e comunidade

- **Issues:** Para dúvidas, sugestões ou proposta de nova trilha
- **Pull Requests:** Para enviar conteúdo ou melhorias
- **Reuniões:** A cada 15 dias — fique de olho nos avisos do repositório
